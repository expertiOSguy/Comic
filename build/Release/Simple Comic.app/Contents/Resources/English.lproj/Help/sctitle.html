Navigation

Page view click
Clicking in the left half of the page view will cause a left page turn, clicking in the right half a right page turn
Option page view click
Clicking in the page view will cause an offset page turn in two page mode, moving only a single page forward instead of the default two
Drag scrolling
Click and drag anywhere in the page view
Arrow keys
Used to scroll a comic view that is too large to display in a window. In addition scrolling to the right or left extreme of a page will cause a page turn to occur
Shift arrow keys
Holding the shift key will triple the default keyboard scrolling speed
Spacebar
Page down, when the page view is at its lowest extreme a forward page turn will occur
Shift spacebar
Page up, when the page view is at its upper extreme a backward page turn will occur
Command right/left arrow
Right or left page turn
Option right/left arrow
Offset page turn in two page mode, moving only a single page forward or backwards instead of the default two
Home and end keys
Go to the first and last page of the comic
Command T
Toggles thumbnail view

Layout

Command D
Change page order from left to right to right to left
This is applicable in both single and two page layout modes
Command P
Change between single and two page layout mode
Commmand L/R
Rotate the page view right or left

Scaling

Command 1
Images is not scaled
Command 2
Images are scaled to fit in the window
Command 3
Images are scaled to fit within the window's horizontal dimension

Zoom

Command =
Zooms in
Command -
Zooms out
Command 0
Resets image zooming

Other

Command F
Enter fullscreen mode
Command U
Toggle the image loupe
Drag files into the view area
Drag files into an existing session's view to open those files within that session
Option key while moving the scroll-wheel horizontally
Change the loupe's diameter
Option key while moving the scroll-wheel vertically
Change the loupe's magnification
